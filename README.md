# Features:
- Sign up, log in, log out
- Create posts
- Like and unlike posts
- Toggle dark/light mode

# How to run locally:
1. Clone the code
3. Create a .env file with MONGO_URL, JWT_SECRET, and PORT variables
4. cd into /server
5. run "npm install"
6. run "npm run start"
7. cd into /client with a new terminal
8. run "npm install"
9. run "npm run start"
